# Zorvyn — Financial dashboard

All assets sit **in the project root** (no `css/` or `js/` folders):

| File | Contents |
|------|----------|
| `index.html` | Page structure only |
| `styles.css` | All styles |
| `app.js` | State, charts, filters, roles, exports |

Open `index.html` in a browser. Keep the three files in the **same folder** so `styles.css` and `app.js` load correctly.
